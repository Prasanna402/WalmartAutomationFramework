# Walmart Automation Framework

## Project Overview
This project is a Hybrid Test Automation Framework built using Selenium WebDriver, Cucumber (BDD), TestNG, and Maven.  
It automates key e-commerce functionalities such as product search and adding products to the cart.

## Tech Stack
- Java
- Selenium WebDriver
- Cucumber (BDD)
- TestNG
- Maven
- Git

## Framework Design
The framework follows the **Page Object Model (POM)** design pattern to improve code maintainability and reusability.

### Components:
- Feature Files (BDD scenarios)
- Step Definitions
- Page Classes
- BaseTest (Driver setup)
- TestRunner (Execution)
- Config file

## Test Scenarios
1. Search Product  
2. Add Product to Cart
